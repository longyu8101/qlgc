8 风荷载
===============

.. raw:: html

  <h2 id="test111">8 风荷载</h2>

8.1 风荷载标准值及基本风压
---------------------------------

.. raw:: html

 <p style="text-align:justify;"><a href="#id8.1.1">8.1.1</a> <span id="id8.1.1">垂直于建筑物表面上的风荷载标准值，应按下列规定确定：</span></p>
 <ol>
 <li>计算主要受力结构时，应按下式计算：</li>
 </ol>

$$w_{k}=β_{z}μ_{s}μ_{z}w_{0}\\tag{8.1.1-1}$$

.. raw:: html
 
 <table border="0" style="font-family:times new roman" id="gongshi">
 <tr>
 <td width="50px" align='center' id="eqzs">式中</td>
 <td width="30px" align='left' id="eqzs"><i>w</i><sub>k</sub></td>
 <td width="40px" align='left' id="eqzs">——</td>
 <td id="eqzs">风荷载标准值（kN/㎡）；</td>
 </tr>
 <tr>
 <td id="eqzs"></td>
 <td id="eqzs"><i>β</i><sub>z</sub></td>
 <td id="eqzs">——</td>
 <td id="eqzs">高度z处的风振系数；</td>
 </tr>
 <tr>
 <td id="eqzs"></td>
 <td id="eqzs"><i>μ</i><sub>s</sub></td>
 <td id="eqzs">——</td>
 <td id="eqzs">风荷载体型系数；</td>
 </tr>
 <tr>
 <td id="eqzs"></td>
 <td id="eqzs"><i>μ</i><sub>z</sub></td>
 <td id="eqzs">——</td>
 <td id="eqzs">风压高度变化系数；</td>
 </tr>
 <tr>
 <td id="eqzs"></td>
 <td id="eqzs"><i>w</i><sub>0</sub></td>
 <td id="eqzs">——</td>
 <td id="eqzs">基本风压（kN/㎡）。</td>
 </tr>
 </table>
 <p></p>
 <ol start="2">
 <li>计算围护结构时，应按下式计算：</li>
 </ol>

$$w_{k}=β_{gz}μ_{sl}μ_{z}w_{0}\\tag{8.1.1-2}$$

.. raw:: html
 
 <table border="0" style="font-family:times new roman" id="gongshi">
 <tr>
 <td width="50px" align='center' id="eqzs">式中</td>
 <td width="30px" align='left' id="eqzs"><i>β</i><sub>gz</sub></td>
 <td width="40px" align='left' id="eqzs">——</td>
 <td id="eqzs">高度z处的阵风系数；</td>
 </tr>
 <tr>
 <td id="eqzs"></td>
 <td id="eqzs"><i>μ</i><sub>sl</sub></td>
 <td id="eqzs">——</td>
 <td id="eqzs">风荷载局部体型系数。</td>
 </tr>
 </table>
 <p></p>
 <p style="text-align:justify;"><a href="#id8.1.2">8.1.2</a> <span id="id8.1.2">基本风压应采用按本规范规定的方法确定的50年重现期的风压，但不得小于0.3kN/㎡。对于高层建筑、高耸结构以及对风荷载比较敏感的其他结构，基本风压的取值应适当提高，并应符合有关结构设计规范的规定。</span></p>
 <p style="text-align:justify;"><a href="#id8.1.3">8.1.3</a> <span id="id8.1.3">全国各城市的基本风压值应按本规范附录E中表E.5重现期R为50年的值采用。当城市或建设地点的基本风压值在本规范表E.5没有给出时，基本风压值应按本规范附录E规定的方法，根据基本风压的定义和当地年最大风速资料，通过统计分析确定，分析时应考虑样本数量的影响。当地没有风速资料时，可根据附近地区规定的基本风压或长期资料，通过气象和地形条件的对比分析确定；也可比照本规范附录E中附图E.6.3全国基本风压分布图近似确定。</span></p>
 <p style="text-align:justify;"><a href="#id8.1.4">8.1.4</a> <span id="id8.1.4">风荷载的组合值系数、频遇值系数和准永久值系数可分别取0.6、0.4和0.0。</span></p>
 <p style="text-align:justify;"><a href="#id8.1.5">8.1.5</a> <span id="id8.1.5">0.5、0.2和0的规定采用。</span></p>

 <p style="text-align:justify;"><a href="#id8.2.2">8.2.2</a> <span id="id8.2.2">对于山区的建筑物，风压高度变化系数除可按平坦地面的粗糙度类别由本规范<a href="#B8.2.1">表8.2.1</a>确定外，还应考虑地形条件的修正，修正系数η应按下列规定采用：</span></p>
 <ol>

 </ol>

$$η_{B}=[1+k\\tanα(1-\\frac{z}{2.5H })]^{2}\\tag{8.2.2}$$

---------------------------------
.. raw:: html

 <li>对于重要且体型复杂的房屋和构筑物，应由风洞试验确定。</li>

8.5 横风向和扭转风振
---------------------------------
.. raw:: html

 <p style="text-align:justify;"><a href="#id8.5.1 ">8.5.1 </a> <span id="id8.5.1 ">对于横风向风振作用效应明显的高层建筑以及细长圆形截面构筑物，宜考虑横风向风振的影响。</span></p>
 <p style="text-align:justify;"><a href="#id8.5.2 ">8.5.2 </a> <span id="id8.5.2 ">横风向风振的等效风荷载可按下列规定采用：</span></p>
 <ol >
 <li style="text-align:justify;">对于平面或立面体型较复杂的高层建筑和高耸结构，横风向风振的等效风荷载<i>w</i><sub>Lk</sub>宜通过风洞试验确定，也可比照有关资料确定；</li>
 <li style="text-align:justify;">对于圆形截面高层建筑及构筑物，其由跨临界强风共振（旋涡脱落）引起的横风向风振等效风荷载<i>w</i><sub>Lk</sub>可按本规范附录H.1确定；</li>
 <li style="text-align:justify;">对于矩形截面及凹角或削角矩形截面的高层建筑，其横风向风振等效风荷载<i>w</i><sub>Lk</sub>可按本规范附录H.2确定。</li>
 </ol>

 <p style="text-align:justify;"><a href="#id8.5.3 ">8.5.3 </a> <span id="id8.5.3 ">对圆形截面的结构，应按下列规定对不同雷诺数<i>R</i><sub>e</sub>的情况进行横风向风振（旋涡脱落）的校核：</span></p>
 <ol >
 <li style="text-align:justify;">当<i>R</i><sub>e</sub> < 3x105且结构顶部风速<i>v</i><sub>H</sub>大于<i>v</i><sub>cr</sub>时，可发生亚临界的微风共振。此时，可在构造上采取防振措施，或控制结构的临界风速<i>v</i><sub>cr</sub>不小于15m/s。</li>
 <li style="text-align:justify;">当<i>R</i><sub>e</sub>≥3.5x106且结构顶部风速<i>v</i><sub>H</sub>的1.2倍大于<i>v</i><sub>cr</sub>时，可发生跨临界的强风共振，此时应考虑横风向风振的等效风荷载。</li>
 <li style="text-align:justify;">当雷诺数为3x105≤<i>R</i><sub>e</sub> < 3.5x106时，则发生超临界范围的风振，可不作处理。</li>
